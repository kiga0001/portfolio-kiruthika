# portfolio-kiruthika
Kiruthika Ganeshan - Portfolio website.
 
## Setup
Prerequisites:
- A modern web browser
- Optional: a local web server (VS Code Live Server)
 
Steps:
1) Open `index.html` directly in your browser, or
2) Open with VS code live server
 
## Project structure
- `index.html` - Main page
- `styles/style.css` - Site styles
- `js/` - JavaScript
- `assets/` - Images
